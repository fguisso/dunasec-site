# DunaSec 2026

Primeiro evento de ciberseguranca do Nordeste.

- **Data:** 30 de maio de 2026
- **Local:** Natal, RN
- **Participantes:** ~250

## Stack

- React 19 + TypeScript
- Vite 7
- CSS Modules
- Deploy via GitHub Pages

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
