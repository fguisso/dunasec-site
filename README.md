# DunaSec 2026

Site da primeira edição do DunaSec — conferência de cibersegurança do Nordeste.

**30 de maio de 2026 · Hotel Barreira Roxa, Natal/RN · 250 participantes**

O evento tem duas trilhas simultâneas: **Técnica**, para devs e analistas, e **Gerencial**, voltada para C-Levels e tomadores de decisão.
